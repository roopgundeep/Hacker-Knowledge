Welcome to Hack-Knowledge

This is Hack-Knowledge, a webapp through which you can teach a class. If you sign in as a teahcer you will be able to perform various function like you will be able to play and pause the video any time and write on the text-pad. All the change that you will do will be synced with with all the other viewers in real time. If you pause the video, it will get paused everywhere. If you are student you will be able to see all these changes. If you join later then u will get synced with teacher's interface at a real time.All the user can chat with each other and clarify there doubts.
Enjoy Learning.
